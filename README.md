### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人信息查看
4. 修改密码
5. 安全退出


**管理员功能：**
1. 登录
2. 系统首页
3. 用户管理
4. 班级管理
5. 学生管理
6. 成绩管理
7. 修改密码
8. 安全退出
